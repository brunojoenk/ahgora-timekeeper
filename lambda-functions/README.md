## Lambda Functions
Funções lambda para utilização na AWS

- [wakeup-heroku-app](wakeup-heroku-app) - Função para subir o servidor Heroku

### Usage

- Gerar arquivo zip
```bash
make build
```
- Subir arquivo zip na AWS ao criar o lambda
